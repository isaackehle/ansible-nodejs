# Ansible: nodejs

Install NodeJS on a system

Available on Ansible Galaxy: [pgkehle.nodejs](https://galaxy.ansible.com/pgkehle/nodejs)


# Examples

```YAML

  - hosts: all
  
    roles:
      - pgkehle.nodejs
```
